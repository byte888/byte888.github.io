# Portfolio Page

A clean, systems-focused portfolio page for positioning around production systems,
internal operational tools, AI-assisted workflows, backend/data pipelines, and deployment
experience.

## Preview

Open `index.html` directly in a browser, or serve the folder locally:

```bash
cd ~/portfolio
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.
